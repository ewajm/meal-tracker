# _Meal Tracker_

#### By _Ewa Manek_

## Description

_Your Eating Habits Are Bad And You Should Feel Bad, a meal-tracker in Angular 2_

## Setup/Installation Requirements

* _Copy the repository from GitHub_
* _Run npm install_
* _run bower install_
* _run gulp serve to build the project for the first time and start the development server_
